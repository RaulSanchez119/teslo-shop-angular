Acerca de Teslo Shop

Teslo Shop es una aplicación de demostración que simula la tienda oficial de merchandising de Tesla. Desde este panel puedes gestionar el catálogo de productos, actualizar precios, controlar el inventario y administrar las imágenes de cada artículo.

Utiliza el menú lateral para navegar entre las secciones disponibles. Accede a Productos para ver y editar el catálogo completo.
